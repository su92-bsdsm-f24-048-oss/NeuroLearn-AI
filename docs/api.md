# API Structure

## Auth
POST /api/auth/register  
POST /api/auth/login  

## Products
GET /api/products  
GET /api/products/:id  

## Cart
POST /api/cart  
GET /api/cart/:userId  

## Recommendations
GET /api/recommendations/:userId  

## Admin
POST /api/products  
PUT /api/products/:id  
DELETE /api/products/:id
